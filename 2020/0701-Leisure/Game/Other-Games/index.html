Ball Sort Puzzle

基础规则

  • 保留一个空格。（不一定要用完空管子，可以把旧管子堆满）
  • 执行绝对优的状态。绝对优：不减少空管子；不减少可能性；连续大于乱序（重点不在连续，而在不同颜色片段数）。
  • 挪出来的有地方挪，新激活的有一点用。

大局观

  • 应用等价思想：如果球在这我会不会挪。例：下图中二层的棕色可以挪出来
    ![[–ballsortpuzzle-LEVEL 179.png]]
  • 想的时候可以忽略目前正在思考的颜色，看看会怎么样。生活适用。
  • 有两个选择的时候，先想想哪个更难（压在下面的更多）
  • 就是整理。不断增加连续同色的数量必能获胜。
  • 可行的必要非充分条件： 选取两种颜色后可以腾出来一个空间，三个颜色两个管子，etc。因此，连续的小球，如果不考虑数量，可以当作一个整体处理。全部是3-1和2-2或1-3没有本质上的不同。此时只需至多一个空管子就能排好。如果每个管子只有3部分，只需要2个管子（猜测，因为要考虑数量）。应用：都是3色只有一个4色管子时，想办法挪4色的。119关验证了。反例（没有空管子）
    ![[–ballsortpuzzle-LEVEL 127.png]]
  • 管子的利用率高对局势的帮助不大，但是如果利用率低的话危害很大。

三个策略

  • 按小球组，这种思路最大的好处，就是保持每个管子的利用率很高：每次交换一组小球，玩的时候讲一个故事，一个小球交换位置，找它应该去的地方。之后长大成两个（或：坐车的时候能不能带个小朋友/小朋友们，你们先出来一下让别人先进去），之后两个两个换。两个同色和其他同色小球难相连，就独立出来，我觉得像孤儿。最后总能把它们连在一起，就像"总会有人来救孤儿的"。
  • 按行（横排）：先把前两层中，每一列的颜色统一，降低复杂度，和第一种（按小球组）很像。
  • 按列（管子）：每次清空一个管子，找管子策略有二：1.（快但不准）找目前最上面一层颜色的小球在哪个管子里最多。2.（准但不快）看有哪些颜色还能放下新球，从每列中排除这些颜色，看剩下的；排除颜色是化简问题，也应该注意数量。前150关用的是这个策略。这种策略是最强的。两个图为例
    ![[–ballsortpuzzle-LEVEL 117.png]]
    ![[–ballsortpuzzle-LEVEL 115.png]]

开局小技巧

  • 如果第一排三个下面不同色只接挪；
  • 找颜色链条，尽量让上两分球同色

其他

  • 重写游戏：可以用数字代替，可以尝试AI
  • 思考：AI怎么教人玩游戏。
